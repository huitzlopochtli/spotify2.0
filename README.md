## Getting Started
### First Add Environment Variables

```bash
NEXT_PUBLIC_CLIENT_ID=""
NEXT_PUBLIC_CLIENT_SECRET=""
NEXT_PUBLIC_JWT_SECRET=""
```
### Then, run the development server:

```bash
npm run dev
# or
yarn dev
```
